Dynamic Programming입니다.


    문제


    Item 1의 이윤 : 4 / 무게 : 1
    Item 2의 이윤 : 5 / 무게 : 3
    Item 3의 이윤 : 6 / 무게 : 4
    최대 무게 : 9

    가방에 최대 무게 9가 넘지 않도록 Item들을 0개 혹은 1개만 넣을 수 있을 때,
    최대 이익이 되도록 넣을 수 있는 방법은?
   

    풀이

    solution을 표로 나타내보고 ‘무게’ 열과 ‘아이템 항목’ 행으로 벡터 S로 구성해보았다.
   
    S[3][9] = max ( S[2][9], S3 + S[2][5] )
    S[2][9] = 9 / S3 + S[2][5] = 15
    S[2][5] = max ( S[1][5], S2 + S[1][2] )
    S[1][5] = 4 / S2 + S[1][2] = 9
    S[1][2] = max ( S[0][2], S1 + S[0][1] )
    S[0][2] = 0 / S1 + S[0][1] = 4
    따라서 solution vector = (1, 1, 1)
